🖥️Батник для удаления следов когда либо запускавшихся программ на пк.
🛡️Скрипт запрашивает права администратора и перезапускает проводник, затем выполняет ряд действий связанных с методом удаления ключей реестра, чистит папки prefetch, Recent, сбрасывает журнал файловой системы (USN Journal), стирая даже упоминания о том, что файлы когда-то были на диске.
🎭Пока идет очистка, окно маскируется под процесс zapret-winws.
🥷После успешной очистки скрипт самоуничтожается и не остается в логах системы.

  ЗАПУСКАТЬ ОТ ИМЕНИ АДМИНИСТРАТОРА
⚠️Не рекомендуется использовать на проверке, скрипт лишь для того, чтобы удалить информацию о ранее запущенных файлах.⚠️
